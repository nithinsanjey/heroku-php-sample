Why jQuery?

March 21, 2015

Probably there are millions of sites and blogs listing the advantages of jQuery. But in this article I would like to share why I use jQuery and how it helps me out.

I initially started up with learning HTML and CSS. Then I continued with javascript. After learning javascript and stepping into jQuery, everything was so simple that it was like cheating. But later I understood that it was the main objective of jQuery "write less, do more."

What is jQuery?

jQuery is a javascript library which has many functions that can be loaded and used. In easy words, jQuery is a javascript program that has functions for many useful javascript implementations that can be made use by including jQuery. It makes writing javascript much easier. jQuery as the name indicates is used for 'querying' for a particular or set of elements.

For example if we want to find the width of an element using javascript, first we need to select the element which can be done by

var myElement=document.getElementById("myEle");

But in jQuery it is like,

var myElement=$("#myEle");

And yes, as you can see from the above example, jQuery uses CSS selectors for querying.

Now let us see a few awesome things about jQuery for which I use it.

1) Simple Querying using CSS selectors.

As seen above jQuery uses CSS selectors for selecting elements we need which makes it easy for going with simple selection of an element to a group of elements. For example,

$("#myEle"); //selects an element of id myEle

$(".myGrp"); //selects elements of class myGrp

$("p"); //selects all the p(paragraph) elements

$("ul li"); //selects all the li elements in all ul(unordered lists)

$("ul#fav li"); //selects all li in a ul of id fav

Pretty cool, right.

2) Cross browser compatibility.

Javascript programs for certain implementations are not all same across different browsers. But with jQuery you need not worry about cross-browser working of your code as jQuery handles that for you.

3) Animation.

jQuery provides us with its own animation functions such as slideDown(), slideUp(), fadeIn(), fadeOut(), …etc an it also provides a animate() function for our own custom animations. For example,

("#myDiv").fadeOut().fadeIn();

is a simple one liner for making the element with id “myDiv” fade out and fade in consecutively. Using this with a continuous loop we can make a continuous fade out and fade in of the element. For example,

setTimeOut(function(){

$("#myDiv").fadeOut().fadeIn();

},1000);

makes the "myDiv" element to fade out and fade in every one second.

4) AJAX.

AJAX (Asynchronous Javascript and XML) is a very useful method by which we can make a request to a server without reloading the page. jQuery provides a function $.ajax() for the implementation of AJAX and it is very handy in any web project.
